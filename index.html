# 一剪梅舟过吴江

一片春愁待酒浇。

江上舟摇，楼上帘招。

秋娘度与泰娘娇。风又飘飘，雨又萧萧。

何日归家洗客袍？

银字笙调，心字香烧。

流光容易把人抛。红了樱桃，绿了芭蕉。

# 虞美人听雨

少年听雨歌楼上，红烛昏罗帐。

壮年听雨客舟中，江阔云低断雁叫西风。

而今听雨僧庐下，鬓已星星也。

悲欢离合总无情，一任阶前点滴到天明。

# 杏帘在望

杏帘招客饮，在望有山庄。

菱荇鹅儿水，桑榆燕子梁。

一畦春韭绿，十里稻花香。

盛世无饥馁，何须耕织忙。
